## 🚀 ローカル開発

### 必要な環境

- Python 3.8以上
- pip

### セットアップと起動

```bash
# 依存関係のインストール
pip install mkdocs-material

# ローカルサーバーの起動
mkdocs serve
```

ブラウザで http://127.0.0.1:8000/ にアクセスしてください。

### ビルド

```bash
mkdocs build
```
